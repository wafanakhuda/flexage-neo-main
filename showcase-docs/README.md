## Quick Start

FlexAGE is a modern web application for flexible assessment and grading. It consists of:

- **Backend**: FastAPI application with PostgreSQL database
- **Frontend**: Next.js React application with TypeScript
- **Key Features**: Assessment creation, student submissions, AI-powered grading

For detailed information, please refer to the individual documentation files.
